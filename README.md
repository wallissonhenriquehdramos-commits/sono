# 💤 Sono XAI — Aplicativo Streamlit

Este app classifica **estágios do sono** (W, N1, N2, N3, REM) com **IA interpretável**.

## 🚀 Funcionalidades
- ✅ Extração de fusos e complexos K (simplificado)  
- ✅ Agrupamento N1+N2 como "Sono Leve"  
- ✅ Relatório + matriz de confusão  
- ✅ Exportação em PDF consolidado  

## ▶️ Como rodar localmente
```bash
pip install -r requirements.txt
streamlit run app.py
